I have built an interactive Tableau dashboard using Global Superstore Excel data by joining Orders and Returns to analyse sales performance and return impact. I have defined key metrics such as category-wise, market-wise, and month-wise sales, along with return-wise sales analysis. Using visualizations like bar, area, pie, and scatter charts, I have designed a dashboard that enabled stakeholders to track trends, compare markets, and identify high-return or high-performing segments, supporting data-driven decisions to improve overall business growth.
